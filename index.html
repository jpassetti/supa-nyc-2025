How to Set Up a Local p5.js Environment in VS Code

  1. Open VS Code and create a new folder for your project (e.g., my-p5-sketch).
  2. Inside your project folder, create a new file named index.html.
  3. Download the latest p5.js library from the p5.js Download page. Save p5.js (or p5.min.js) into your project folder.
  4. In your index.html, add a <script> tag that links to your local p5.js file:
    <script src="p5.js"></script>
  5. Create a new file named sketch.js in the same folder. Write your p5.js code there.
  6. In index.html, add another <script src="sketch.js"></script> tag after the p5.js script tag.
  7. Open index.html in your browser to run your sketch. You can use the "Go Live" extension in VS Code for easy local preview.
  8. For more help, see the p5.js Get Started Guide.

Tip: You can also use the online p5.js Web Editor for quick experiments.

How to Set Up Processing.py (Python Mode)

  1. Download and install Processing (the Java-based IDE) for your OS.
  2. Open Processing, then go to Tools → Manage Tools..., click on the Modes tab and look for Python Mode for Processing 4. Click "Install".
  3. Switch to Python Mode using the dropdown in the top-right corner of the Processing window.
  4. Create a new sketch and paste in the Processing.py code.
  5. Click the "Run" button (triangle) to run your Python sketch!
  6. For more help, see the Processing.py Getting Started Guide.

Note: Processing.py is not the same as p5py or p5.js. It runs in the Processing IDE and uses Python syntax to control the Processing graphics engine.
