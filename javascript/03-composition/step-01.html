Step 1: 12x12 Tile Grid Composition

Teacher Notes:
This step demonstrates how to use nested loops to create a grid and introduce randomness in generative art. At each grid point, a random black circle, square, or line is drawn, all the same size and with no rotation. This is a great way to discuss iteration, randomness, and visual constraints in creative coding.

Classroom Tips:
- Ask students to predict what will happen if you change the grid size or shape types.
- Discuss how constraints (like using only black) can lead to interesting results.
- Challenge students to modify the code to add color or animation in later steps.
