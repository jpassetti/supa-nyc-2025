Step 2: Color Triad Grid Remix

Remix the grid!
Step 2 creates a playful, abstract grid using a color triad (three colors spaced evenly on the color wheel) and layering. Each cell is filled with either a colored or white square, and may have a half-circle (with a different color) on top, randomly rotated.

How it works (JavaScript logic):

  • The grid is 12x12. Each cell is drawn from the center out, so shapes fill the space with no gaps.
  • Background: Each cell randomly gets a colored square (from the triad) or a white square (blank).
  • Top shape: Each cell randomly gets either a half-circle (with a different triad color, never matching the background if colored) or nothing (blank).
  • The half-circle is randomly rotated 90°, 180°, or 270° for variety.
  • All shapes are drawn with noStroke() for a clean look.

What is a color triad?
A color triad is a set of three colors spaced evenly around the color wheel. This creates strong visual contrast and harmony.

Classroom/Teacher Notes:
  • Discuss how layering and negative space affect the composition.
  • Challenge students to remix: change the triad, grid size, or add new shapes.
  • Ask: What happens if you use analogous or complementary colors instead?
  • Try making all backgrounds white, or all backgrounds colored, for different effects.
  • Encourage students to explain the random logic in their own words.

Code summary:
  • Each cell: rect() for background, arc() for half-circle (if chosen).
  • Random choices ensure every grid is unique on each run.
  • Half-circles never match the colored background for maximum contrast.
