Step 3: Weather Data Grid Visualization

Visualize a year of real weather data!
This step fetches the last 365 days of daily high temperatures for Syracuse, NY from the free Open-Meteo API.
Each day is shown as a circle in a grid. Small blue circles = cold days, large red/orange circles = hot days. The color spectrum blends smoothly from blue (cold) to red (hot), like a weather map.

How it works (for beginners):

  • The code uses JavaScript and the p5.js library to draw on a canvas.
  • It uses the fetch() function to get real weather data from the Open-Meteo API (no login or key needed).
  • The API returns a list of 365 daily high temperatures for Syracuse, NY, in Fahrenheit.
  • The code arranges these 365 days in a grid (20 columns x 19 rows) so every day fits on the canvas.
  • Each circle's size and color are mapped to the temperature for that day: colder = smaller and blue, hotter = larger and red/orange.
  • The color smoothly transitions through blue, cyan, green, yellow, orange, and red, just like a weather map.
  • If the data is still loading, you'll see a loading message. If the API fails, you'll see an error message.

What is an API?
An API (Application Programming Interface) lets you get data from another website or service. Here, we use the Open-Meteo API to get real weather data for free.

Classroom/Teacher Notes:
  • Discuss how data can be mapped to visual properties (size, color, position).
  • Challenge students to use other weather data (rainfall, low temps, etc) or try a different city by changing the latitude/longitude in the code.
  • Ask: How does the color mapping affect your perception of the data? What other color palettes could you use?
  • Try changing the grid size, color stops, or mapping for new effects.
  • Encourage students to explain the code logic in their own words, especially how the API data is used.

Remix ideas:
  • Change the city by editing the latitude/longitude in the code.
  • Visualize a different weather variable (like rainfall or low temperature).
  • Try a different color palette or shape for the data points.
  • Animate the data over time, or add tooltips for each day.
