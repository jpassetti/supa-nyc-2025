Step 3: Perlin Noise Demo (2D)

Code is available on the Github repository.

Teacher Notes:
This step uses two dimensions of Perlin noise to move a circle smoothly in both x and y. This creates a floating, natural motion.

Classroom Tips:
- Ask students to describe the path of the circle.
- Discuss how 2D noise can be used for clouds, terrain, or movement.
- Try changing the speed or range of movement.

The circle now floats around the canvas in two dimensions.
