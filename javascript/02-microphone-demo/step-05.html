Step 5: Microphone Color Lerp

Code is available on the Github repository.

Teacher Notes:
This step uses the microphone to control both the size and color of a circle.
The louder the sound, the bigger the circle gets, and the color shifts from blue (quiet) to magenta (loud).

How the Sound Data is Used:
- The computer listens to the microphone and measures the "amplitude" (loudness) of the sound in real time.
- The code averages the absolute value of the audio waveform to get a number between 0 (quiet) and about 0.5 (very loud).
- This amplitude value is used in two ways:
1. Size: The louder the sound, the larger the circle. (Amplitude is mapped to a diameter from 80 to 700 pixels.)
2. Color: The amplitude is also used to blend between two colors: blue for quiet, magenta for loud.
- The blending is done with a function called lerpColor().

What does "Lerp" mean?
- "Lerp" stands for Linear Interpolation.
- It means to smoothly mix between two values. In this case, it mixes between the blue color and the magenta color.
- When the sound is quiet, the circle is all blue. As the sound gets louder, the color gradually shifts toward magenta.
- This is a common technique in animation and creative coding to create smooth transitions.

How to Use:

Classroom Tips:
- Ask students to observe how the color and size change with volume.
- Try mapping the mic input to other properties (e.g., position, shape).
- Discuss how "lerp" can be used for smooth transitions in other creative projects.
