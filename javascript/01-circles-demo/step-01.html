Step 1: Draw One Circle

Teacher Notes:
This step introduces the basics of drawing a single shape in p5.js. Use this to discuss the coordinate system, the ellipse function, and how code translates to visual output.

Classroom Tips:
- Ask students to change the position or size of the circle.
- Discuss the difference between x/y coordinates and diameter.
- Use this as a starting point for exploring other shapes.

ellipse(x, y, diameter);
