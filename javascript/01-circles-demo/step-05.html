Step 5: Animated Grid

Code is available on the Github repository.

Teacher Notes:
This step introduces animation and interactivity. Students see how variables and update logic can create movement, and how to save their generative art.

Classroom Tips:
- Ask students to change the animation speed or range.
- Discuss how animation can be used for both art and data visualization.
- Try adding sliders or buttons for more interactivity.

Press S to save an image to your computer (usually the Downloads folder).
