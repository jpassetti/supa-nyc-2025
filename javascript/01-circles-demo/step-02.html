Step 2: Row of Circles

Teacher Notes:
This step introduces the concept of iteration using a for loop to draw multiple shapes. Students see how code can automate repetitive tasks and create patterns.

Classroom Tips:
- Ask students to change the number of circles or their spacing.
- Discuss how the loop variable controls position.
- Try changing the shape or color for each circle.

for (let i = 0; i < n; i++) { ellipse(...); }
