Step 4: Grid with Randomness

Code is available on the Github repository.

Teacher Notes:
This step introduces randomness in both size and color. Students see how generative art can be made more dynamic and expressive by using random values.

Classroom Tips:
- Ask students to experiment with the range of random values.
- Discuss how color and size affect the visual impact.
- Try using random() for position or other properties.
