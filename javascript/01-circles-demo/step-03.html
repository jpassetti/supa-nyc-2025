Step 3: Grid of Circles

Code is available on the Github repository.

Teacher Notes:
This step demonstrates nested loops to create a 2D grid. Students see how two loops can control both x and y positions, building more complex patterns.

Classroom Tips:
- Ask students to change the number of rows/columns.
- Try using different shapes or colors in the grid.
- Discuss how nested loops relate to tables or spreadsheets.

for (let i = 0; i < cols; i++) { for (let j = 0; j < rows; j++) { ellipse(...); } }
