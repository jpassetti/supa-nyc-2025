Step 2: Color Triad Grid Remix (Processing.py)

Code is available on the Github repository.

Create a colorful grid composition using a color triad and random layering!
This step matches the JavaScript demo, but uses Python code in Processing's Python mode.

Python code for this step:
py/step-02.py

How it works:

  • Each cell is filled with either a colored or white square.
  • On top, a half-circle (with a different color) may be drawn, randomly rotated.
  • Some cells are left blank on top, creating negative space.
  • Uses a color triad for strong visual contrast and harmony.

Instructions:
  • Open py/step-02.py in the Processing IDE (Python mode).
  • Click the Run button (triangle) to see the composition.
  • Try changing the triad, grid size, or add new shapes!

Teacher Notes:
  • This sketch is a direct translation of the p5.js version for easy comparison.
  • Use this to discuss color theory, layering, and abstraction.
  • Challenge students to remix: change the triad, grid size, or add new shapes.
