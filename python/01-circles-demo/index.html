Demo 1: Circle Grid with Randomness (Processing.py)

Code is available on the Github repository.

This demo series recreates the JavaScript/p5.js circle grid project using Python in Processing.py.
Each step has a Python file and a classroom-ready HTML page with teacher notes and instructions.

  1. Step 1: Draw One Circle
    - Draw a single circle at the center of the canvas
    - Introduces ellipse(), fill(), and the coordinate system
    - Python code: py/step-01.py
  2. Step 2: Circle Grid with Loops
    - Draw a grid of circles using two for-loops (no randomness yet)
    - Great for introducing loops and coordinate systems
    - Python code: py/step-02.py
  3. Step 3: Add Randomness to Size and Color
    - Make each circle a different size and color using random()
    - Great for exploring generative art and randomness
    - Python code: py/step-03.py
  4. Step 4: Animate the Circles
    - Animate the circles with a pulsing effect using draw() and sin()
    - Introduces animation and time-based art
    - Python code: py/step-04.py
  5. Step 5: Save Your Art
    - Save your generative art as an image by pressing a key or clicking the canvas
    - Great for sharing or printing your creations
    - Python code: py/step-05.py
