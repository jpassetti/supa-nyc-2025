Step 5: Save Your Art (Processing.py)

Save your generative art as an image file!
This step matches the JavaScript demo, but uses Python code in Processing's Python mode.

Python code for this step:
python/01-circles-demo/py/step-05.py

How it works:

  • Uses saveFrame() to save the canvas as an image file.
  • Click the canvas or press a key to save your artwork.
  • Great for sharing or printing your generative creations!

Instructions:
  • Open py/step-05.py in the Processing IDE (Python mode).
  • Click the Run button (triangle) to see the art.
  • Click the canvas or press a key to save your image.

Teacher Notes:
  • This sketch is a direct translation of the p5.js version for easy comparison.
  • Use this to discuss digital art, file formats, and sharing work.
  • Challenge students to create a series of generative images.
