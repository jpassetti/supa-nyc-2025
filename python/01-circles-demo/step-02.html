Step 2: Circle Grid with Loops (Processing.py)

Code is available on the Github repository.

Draw a grid of circles using Python and Processing.py!
This step matches the first JavaScript demo, but uses Python code in Processing's Python mode.

Python code for this step:
python/01-circles-demo/py/step-02.py

How it works:

  • Uses two for loops to create a grid (rows and columns).
  • Each circle is the same size and color (no randomness yet).
  • Great for learning about loops and coordinate systems in creative coding.

Instructions:
  • Open py/step-02.py in the Processing IDE (Python mode).
  • Click the Run button (triangle) to see the grid of circles.
  • Try changing the number of rows/columns, or the circle size!

Teacher Notes:
  • This sketch is a direct translation of the p5.js version for easy comparison.
  • Use this to introduce students to Python syntax and Processing's graphics functions.
  • Discuss similarities and differences between Processing.py and p5.js.
