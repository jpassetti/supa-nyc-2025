Step 4: Animate the Circles (Processing.py)

Bring your generative art to life with animation!
This step matches the JavaScript demo, but uses Python code in Processing's Python mode.

Python code for this step:
python/01-circles-demo/py/step-04.py

How it works:

  • Uses draw() to animate the circles over time.
  • Each circle's size or color can change every frame, creating movement.
  • Great for exploring animation and time-based generative art!

Instructions:
  • Open py/step-04.py in the Processing IDE (Python mode).
  • Click the Run button (triangle) to see the animation.
  • Try changing the animation speed or what properties are animated!

Teacher Notes:
  • This sketch is a direct translation of the p5.js version for easy comparison.
  • Use this to discuss animation, frame rate, and creative coding.
  • Challenge students to animate color, position, or other properties.
