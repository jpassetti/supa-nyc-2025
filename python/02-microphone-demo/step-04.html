Step 4: 2D Perlin Noise Island Map (Processing.py)

Code is available on the Github repository.

Generate an island map using 2D Perlin noise!
This step matches the JavaScript demo, but uses Python code in Processing's Python mode.

Python code for this step:
py/step-04.py

How it works:

  • Uses noise(x, y) to generate a value for each grid cell.
  • Colors each cell as ocean, sand, land, or mountain based on the noise value.
  • Great for exploring procedural generation and Perlin noise!

Instructions:
  • Open py/step-04.py in the Processing IDE (Python mode).
  • Click the Run button (triangle) to see the island map.
  • Try changing the thresholds or colors for different effects!

Teacher Notes:
  • This sketch is a direct translation of the p5.js version for easy comparison.
  • Use this to discuss procedural generation, noise, and creative coding.
  • Challenge students to animate the map or add new terrain types.
