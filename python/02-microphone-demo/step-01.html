Step 1: Random Numbers Seismograph (Processing.py)

Visualize randomness as a seismograph and a jumping circle!
This step matches the JavaScript demo, but uses Python code in Processing's Python mode.

Python code for this step:
py/step-01.py

How it works:

  • Uses random() to generate a new value each frame.
  • Draws a line chart (seismograph) and a jumping circle to show the random value.
  • Great for discussing randomness and data visualization!

Instructions:
  • Open py/step-01.py in the Processing IDE (Python mode).
  • Click the Run button (triangle) to see the visualization.
  • Try changing the range or speed of the random values!

Teacher Notes:
  • This sketch is a direct translation of the p5.js version for easy comparison.
  • Use this to discuss randomness, data, and creative coding.
  • Challenge students to visualize other types of data.
