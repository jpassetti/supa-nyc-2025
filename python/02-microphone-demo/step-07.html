Step 7: Microphone Recording to Circles (Processing.py)

Visualize a pre-recorded audio file as a series of circles!
This step matches the JavaScript demo, but Processing.py does not support microphone input natively.

Python code for this step:
py/step-07.py

How it works:

  • Loads a pre-recorded data file (CSV or text) representing audio levels.
  • Draws a circle for each value, visualizing the "recording" as a timeline.
  • Great for exploring data-driven art and audio visualization!

Instructions:
  • Open py/step-07.py in the Processing IDE (Python mode).
  • Place a CSV or text file with audio levels in the sketch folder.
  • Edit the code to match your data file's format if needed.

Teacher Notes:
  • Processing.py does not support real-time mic input out of the box.
  • Use this to discuss data visualization, audio, and creative coding.
  • Challenge students to record their own data and visualize it!
