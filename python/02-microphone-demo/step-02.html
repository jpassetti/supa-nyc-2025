Step 2: Perlin Noise Seismograph (Processing.py)

Visualize Perlin noise as a smooth seismograph and a moving circle!
This step matches the JavaScript demo, but uses Python code in Processing's Python mode.

Python code for this step:
py/step-02.py

How it works:

  • Uses noise() to generate smooth, natural-looking values each frame.
  • Draws a line chart (seismograph) and a smoothly moving circle to show the noise value.
  • Great for discussing randomness vs. noise and data visualization!

Instructions:
  • Open py/step-02.py in the Processing IDE (Python mode).
  • Click the Run button (triangle) to see the visualization.
  • Try changing the speed or scale of the noise!

Teacher Notes:
  • This sketch is a direct translation of the p5.js version for easy comparison.
  • Use this to discuss Perlin noise, randomness, and creative coding.
  • Challenge students to visualize other types of smooth data.
