Step 6: Microphone Color Lerp (Processing.py)

Code is available on the Github repository.

Change the color of a circle based on simulated mic input!
This step matches the JavaScript demo, but Processing.py does not support microphone input natively.

Python code for this step:
py/step-06.py

How it works:

  • Simulates mic input with random values.
  • Uses lerpColor() to blend between two colors based on the simulated input.
  • Great for exploring color interpolation and data-driven art!

Instructions:
  • Open py/step-06.py in the Processing IDE (Python mode).
  • Read the code comments for ideas on how to simulate or extend this demo.

Teacher Notes:
  • Processing.py does not support real-time mic input out of the box.
  • Discuss how to use pre-recorded data, or how to use Java libraries for advanced students.
  • Encourage creative workarounds or data-driven art!
